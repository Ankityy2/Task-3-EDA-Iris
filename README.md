
# Iris Dataset EDA

## Objective
Perform Exploratory Data Analysis (EDA) on the Iris Dataset to understand data patterns and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Data Loading
- Data Inspection
- Missing Value Analysis
- Summary Statistics
- Histogram Visualization
- Boxplot Analysis
- Correlation Heatmap
- Pairplot Analysis

## Key Findings
- No missing values found.
- Petal Length and Petal Width are highly correlated (0.96).
- Setosa species is clearly distinguishable.
- Petal features are more useful than sepal features for classification.
